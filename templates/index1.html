Satellite Tracking API documentation

Endpoints

  • GET /api/active_sat/<name>

    • 			URL: /api/active_sat/<name>
			Method: GET
			description: Retrieve the position and velocity of satellite at current time zone using name of satellite
			Response:
				404:
				description: Not Found
				200:
				description: successful request
				scheme:
				{
                		"a_date": 2024-7-14,
                		"a_time": 2:30:15,
                		"name": STELLA,
                		"r_vector": {"x": 123.45, "y": 678.90, "z": 234.56},
                		"v_vector": {"x": 7.89, "y": 6.54, "z": 3.21}
                		}

  • PUT /api/active_sat/<name>

    • 			URL: /api/active_sat/<name>
			Method: PUT
			description: Update the position and velocity of satellite at specified datetime
			Request:
				{
  				"date_time": "2024-07-14 10:00:00"
				}
			Response:
				404:
				description: Not Found
				200:
				description: successful request
				scheme:
				{
                		"a_date": 2024-7-14,
                		"a_time": 10:00:00,
                		"name": STELLA,
                		"r_vector": {"x": 123.45, "y": 678.90, "z": 234.56},
                		"v_vector": {"x": 7.89, "y": 6.54, "z": 3.21}
                		}
